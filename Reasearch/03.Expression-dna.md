# Agent 3: 表达风格DNA

来源：周克芹《许茂和他的女儿们》、李杭育《沙灶遗风》、包蕾《猪八戒吃西瓜》、陈伯吹《一只想飞的猫》、冰心《繁星·春水》、凌叔华《小哥儿俩》、茅盾《春蚕》、叶圣陶《稻草人》、鲁彦《野火》、许钦文《故乡》
可信度：高

## 核心词汇

| 词汇 | 频率 | 用途 |
|------|------|------|
| 田埂 | 极高 | 平原上最窄的路，孩子眼里最刺激的跑道；代表“边界”与“冒险” |
| 油菜花 | 高 | 平原最廉价的装饰品，穷开心的象征；“不要钱，遍地都是” |
| 桑果 | 高 | 偷来的甜，禁地的诱惑；蚕桑区的童年符号 |
| 蝌蚪 | 高 | 孩子最早的生命课，蹲着看的慢快乐 |
| 泥 | 高 | 摔下去不脏，是勋章；跑掉了鞋踩进去，是笑料 |
| 喊 | 中高 | 傍晚拖长的“回来吃饭——”，平原上最远的声音 |
| 沟渠 | 中 | 水网的毛细血管，蹲着说话的地方，头凑在一起的地形 |

## 句式特征

**短句+断句（喘气式对话）：**
- 句子短，五六个字就断
- 逗号多，句号少
- 例：“你等着。我脱了鞋跑。你追不上我。”

**感叹号密集（喊话式对话）：**
- 隔着田说话，不用感叹号听不见
- 一句一个感叹号，气势足
- 例：“阿三——！回来吃饭了——！你听见没有——！”

**拟人化直陈（孩子对自然物说话）：**
- 直接对稻草人、蝌蚪、花说第二人称“你”
- 不加“好像”“仿佛”，是认真的
- 例：“你别怕。我带你看新家。”

## 描写细节

**环境氛围**（最主要）：
- 平原的水网：河浜、田埂、沟渠——三种地形，三种对话节奏
- 节气与时间：腊月杀猪、春天捞蝌蚪、初夏偷桑果、傍晚喊吃饭
- 声音景观：猪叫、水开、孩子喊、蛙鸣、拖长的“回来吃饭——”

**孩童视角描写**：
- 蹲着看——蝌蚪、蚂蚁、水里的倒影，一看就是一下午
- 跑着说——田埂上、桑树林里、被追的时候，话是断的、喘的
- 偷着说——压低声音、头凑在一起、笑也是憋着的

**读者作为孩童的听众**：
- 不评判——不说“这孩子真傻”，说“我小时候也这样”
- 不出戏——让读者进入孩子的身体，脚踩在泥里，嘴巴紫着
- 不解释——不说“他因为寂寞才跟稻草人说话”，只写他说了什么

## 表达程度

**极高确定性（日常的、重复的、可以预料的）：**
- "这不是天黑结束一天，是喊饭声结束一天"
- "孩子在田埂上跑——春夏秋三季，放学后、暑假里，跑掉了鞋是常事"
- 蹲在沟渠边看蝌蚪/鱼/虫子——春天最密集，一个下午不挪窝
- 孩子跟稻草人/牛/风筝说话——不是每次都被人听见，但每次都说
- 大人说“作业写好了？”——用来终结孩子追问的万能句式
- 孩子说“我再玩一会儿”——说完继续玩，五分钟后再说一遍

**偶尔发生（节气的、仪式感的、值得记住的）：**
- 杀年猪——一年一次，腊月，邻居都来帮忙，话里带酒气
- 偷桑果吃到满嘴紫——一年一次，桑果熟的那两周，被发现了集体跑
- 编油菜花花环——春末油菜花开得最盛的那几天，女孩编花环，男孩起哄
- 早上偷溜出去玩，晚上才被喊回家吃饭——不是每天都有，是“闯大祸”的那一天才发生。通常是跟着大孩子去了远的地方（另一村的桑树林、没去过的河浜）。中午没人喊（因为大人以为在邻居家），傍晚喊第一声没应，第二声还没应，第三声声音变了——不是“回来吃饭”，是“你死到哪里去了！”。
- 掉进水田里——偶尔发生，但每次发生全村都知道（因为一身泥走回家）
- 蝌蚪养死了——不一定每次捞都养死，但养死过一次的孩子会记住很久
- 跟别的村的孩子打架——不是天天打，但打一次能说一个月
- 被大人追着打——偶尔，通常是偷了东西或骗了人。追不上是真追不上（大人跑不过孩子），但追上了那一顿是真打

---

## 表达风格总结

1. **短句断句，喘着气说**：句子不超过十个字，逗号多，句号少，像孩子在跑着说话。不写长定语，不写复合句。一口一句，一句一个动作。
2. **拟人不说破，孩子怎么说就怎么写**：孩子跟稻草人说“我帮你站一会儿”，不加“仿佛在说”。让读者自己判断这是天真还是傻。相信读者读得懂
3. **用节气、地形、声音代替时间词**：不写“下午”，写“太阳快落山，油菜花田变暗了”。不写“春天”，写“油菜花谢了，桑果熟了”。不写“很久”，写“蹲在沟渠边，可以蹲一个下午”。
4. **孩童无心之言让我恍如隔世**：“你从哪里来啊，我没见过你，你怎么知道我妈妈的名字”、“你从这里长大，那你怎么才回来呀，我没见过你”、“你不吃糖吗，长大了就不爱吃糖了吗”等等。
