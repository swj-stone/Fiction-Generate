# Agent 5: 决策与行为模式

来源：童年心理学研究（皮亚杰认知发展阶段）、乡土文学中儿童角色分析（《春蚕》中的少年视角、《草房子》中的桑桑等）
可信度：中高（基于文学典型与儿童发展心理学的交叉分析）

---

## 决策路径

### 1. 被喊回家吃饭时的“拖延决策”
- **背景**：傍晚，母亲喊第三遍了，声音从“回来吃饭——”变成了“你死到哪里去了——”。孩子已经听见了，但正在玩一件舍不得停的事（捞蝌蚪/捉迷藏/看蚂蚁搬家）
- **决策**：先喊一声“哎——来了——”，但身体不动。玩两三分钟，再喊一声“来了来了——”，还是不动。直到听到母亲声音里带了真火，才一身泥跑回去
- **结果**：多数时候平安无事（母亲只是嘴上凶），偶尔被追着打（如果拖太久或身上太脏）
- **逻辑**：孩子的判断依据不是“该不该回去”，而是“母亲的声音到了哪一级”——第一级是通知（可忽略），第二级是催促（可拖延），第三级是警告（必须跑）。孩子通常依依不舍，不想回家，但是又不得不离开

### 2. 蝌蚪养死后的“归因决策”
- **背景**：捞了一瓶蝌蚪，放在窗台上。第二天全死了，水变浑了，蝌蚪沉在瓶底不动。孩子蹲在瓶子前，不说话
- **决策**：孩子会自己找一个“为什么死”的解释——不是“缺氧”（太抽象），而是“它不喜欢瓶子”“我给的水不干净”“我摸它太多了”。然后决定下次要不要捞：有的说“再也不捞了”，有的说“下次换个大的缸”
- **结果**：哭一场。母亲会说“早说了养不活”。孩子记很久。下次路过沟渠，会多看两眼，但不一定伸手
- **逻辑**：孩子从这件事里学到的不是“生物学”，是“喜欢一个东西不等于能留住它”。这是童年最温和的失去教育

### 3. 跟稻草人说话时的“认知决策”
- **背景**：一个人放牛，或一个人在田边等小伙伴。面前有一个稻草人，破草帽、旧衣裳，歪歪地站着。没有别人可以说话
- **决策**：孩子会主动给稻草人“分配角色”——“你帮我看着牛，我去那边看一眼”。或者问稻草人问题：“你热不热？我把帽子给你戴。”然后自己替稻草人回答：“不要，我有帽子。”
- **结果**：玩得很认真，不觉得尴尬。被大人看到了，大人笑：“你跟个稻草人说什么？”孩子答：“它听得懂。”然后跑开
- **教训**：孩子没有“假想”和“真实”的严格边界。稻草人在孩子的认知里是“有空位的角色”——它不说话，正好让我替它说。这不是孤独，是创造

### 4. 捡到别人东西时的“处置决策”
- **背景**：在田埂上捡到一把小刀、一颗弹珠、或者几毛钱。四周没人。这东西自己想要
- **决策**：孩子会有一个短暂的“内心辩论”——“捡到就是我的”vs“别人丢了会找”。多数孩子的第一反应是揣兜里。但晚上会睡不着。第二天要么还回去（说“我捡到的”），要么藏起来不敢用。极少数孩子会直接交给大人
- **结果**：还回去的，对方说“谢谢你”，孩子说“不客气”，但心里有点舍不得。藏起来的，过几天拿出来玩，被认出来了，大人问“哪来的”，孩子说“捡的”，大人说“捡的也是别人的”，然后没收
- **教训**：孩子的道德判断不是“对错”，是“会不会被抓”和“晚上会不会睡不着”。捡到东西藏起来，快乐是“我有了”，痛苦是“这是别人的”。孩子在这两个感觉之间摇摆。这是最早的“所有权”教育

## 决策启发式（从行为模式提炼）

| 情境 | 他的规则 |
|------|---------|
| 面对“该回家但还没玩够” | 分级判断母亲的声音：第一遍“通知”（可忽略），第二遍“催促”（可拖延），第三遍“警告”（必须跑）。跑的时候想好理由：“我在帮阿婆找鸡” |
| 面对“弄坏了东西/犯了错” | 第一反应是“能不能不被发现”。如果能，自己处理（洗、藏、假装没事）。如果不能，选一个“减刑”的认错时机（比如母亲心情好的时候） |
| 面对“跟小伙伴吵架了” | 不道歉，用行动代替：踢一块土疙瘩过去、递半块饼、指着远处说“你看那是什么”。对方接了，就和好了。不说“对不起” |
| 面对“一个人没事做” | 找一个“可以说话的东西”——稻草人、牛、狗、蝌蚪、风筝。给它们分配角色，替它们回答。这是“自创玩伴”，不是孤独 |
| 长期策略 | 在“不被大人发现”和“不被大人打死”之间找平衡。玩要尽兴，但留三成注意力听大人的声音方向。快乐是偷出来的，所以更开心 |

## 言行一致性分析

**一致的：**
- 拖延策略贯穿所有“不想做”的事：喊吃饭时拖延、叫帮忙时拖延、写作业时拖延。孩子用的是同一套“能拖就拖，拖不了就跑”的策略
- 用行动代替语言表达情绪：不和好了不说话，踢土疙瘩；不道歉，递半块饼；不认输，光脚继续跑。孩子的“做”比“说”多
- 对自然物的拟人化是稳定的认知方式：不只是跟稻草人说话，也跟蝌蚪、牛、风筝说话。孩子一直用“万物有灵”的眼睛看世界，不分场合

**存在内在张力的：**
- “想被看见”vs“不想被发现”：孩子希望自己的“英雄行为”（光脚跑、偷到桑果、抓到最大的蝌蚪）被同伴看见并认可，但绝对不想被大人发现。同一件事，在同伴面前是勋章，在大入面前是罪证
- “想长大”vs“不想被当大人用”：孩子模仿大人说话（“今年的蚕不好养”），但真的被叫去帮忙时能躲就躲。想享受“大人的话语权”，不想承担“大人的劳动量”
- “想养蝌蚪”vs“养死了难过”：捞的时候很兴奋，养死了很自责，但第二年又去捞了。想拥有，又承受不了失去。这不是矛盾，是孩子在学“拥有和责任的关系”，只是学得很慢
